# CSharp_POO_GUI_SQLite
Examen de programación en C# en el que tuve que desarrollar una pequeña aplicación para una agencia de viajes.
Visualización y guardado paquetes de viajes mediante el uso de programación orientada a objetos, interfaz generada con Glade y manejo de datos con SQLite.

Lenguaje de programación: C# (entorno .NET)
Librerías: GUI Glade (GTK), SQLite.
Herramientas / IDE: Visual Studio 2019, Glade.
